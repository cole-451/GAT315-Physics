# GAT315-Physics
a basic 2D physics simulator using the Raylib-Quickstart framework, which allows you to spawn bodies, effectors and tweak all of the settings for gravity.

## Controls
Press your left mouse button to spawn a body. Multiple bodies can be spawned if you press the left mouse along with CTRL.
You can spawn effectors by pressing SHIFT along with the left mouse button.
Hold down the right mouse button on a body and release it on another to make a spring.
